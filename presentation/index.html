What is React?

  • Open-source JavaScript library for rendering views
  • Initial release on March 2013 by Jordan Walke, a Facebook Software Engineer
  • Renders HTML written is JS in its own virtual DOM

Why React?

  • Component Oriented development
  • Efficiency taking advantage of the Virtual DOM
  • Allows Server-side rendering
  • Excelent Maintenance
  • Bonus: React Native
